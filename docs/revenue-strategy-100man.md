### CEO分析: ビジョンと成長戦略

#### 現状の問題

**Tanuki Tabi Travelは「優れたガイドサービス」だが「ビジネス」としてはまだ未成熟。**

1. **収益源が1つしかない**: Manabuの身体が資本。体調不良・休暇で収入ゼロ
2. **スケールしない**: 1日1〜2ツアーが物理的上限。月間最大ツアー数 ≒ 30〜40本
3. **予約導線が弱い**: 月間8万impのサイトから、contactページへの到達が月1件（GSC上）
4. **ブランド資産が活かされていない**: 500+ツアー実績、4.86★評価、国家資格ガイドという強力なブランドが、ツアー以外のマネタイズに一切使われていない

#### 月100万円のビジョン

```
ツアー売上:    ¥600,000（月15本 × 平均¥40,000）
アフィリエイト: ¥200,000（JR Pass + ホテル + アクティビティ）
デジタル商品:   ¥100,000（PDF + メール講座）
広告/スポンサー: ¥100,000（ディスプレイ広告 + 記事スポンサー）
────────────────────────────────────
合計:          ¥1,000,000/月
```

#### CEO提言

1. **「ガイドのブランド」から「メディア企業」への転換**: サイトは「ツアーの予約ページ」ではなく「日本旅行の意思決定ハブ」。トラフィックを多角的に収益化する
2. **スケーラブルな収益を優先**: ツアー売上は上限があるが、アフィリエイト・デジタル商品・広告は睡眠中も稼ぐ
3. **3ヶ月以内にツアー以外の収益を¥300,000以上にする**: これでManabuが月に5日休んでも¥700,000以上の収入

### CSO（最高戦略責任者）分析: 競合と差別化

#### 競合マップ

| 競合 | 強み | 弱み | 脅威度 |
|------|------|------|--------|
| **GetYourGuide** | SEO最強、レビュー数万件、即時予約 | 没個性的、ガイド品質不明 | 高 |
| **Viator** | TripAdvisor連携、巨大トラフィック | 同上 | 高 |
| **MagicalTrip** | 日本特化、多数のガイド | 品質にばらつき | 中 |
| **GoWithGuide** | 個人ガイドマッチング | UI/UX弱い | 中 |
| **個人ガイド（同業）** | 専門性、個人ブランド | マーケティング弱い | 低 |

#### Tanuki Tabiの差別化ポイント

1. **国家資格**: 全国通訳案内士 — 日本で最も権威あるガイド資格。競合の95%は無資格
2. **コンテンツ力**: SEOブログがトラフィックを生む。GetYourGuideのアフィリエイター的存在になれる
3. **カスタマイズ**: OTA（GetYourGuide等）では不可能な完全カスタム対応
4. **3ヶ国語**: 英語 + スペイン語 + 日本語。スペイン語対応は希少価値

#### CSO提言: 「コンテンツ → 信頼 → 予約」のフライホイール

```
    ブログ記事（SEO）
         ↓
    読者がサイトに来る
         ↓
    メールアドレスを取得
         ↓
    ウェルカムシーケンスで信頼構築
         ↓
    ツアー予約 or アフィリエイト購入
         ↓
    ツアー後レビュー → Google評価UP
         ↓
    レビュー → 新しい読者の信頼 → 予約
         ↓
    （フライホイールが加速）
```

#### 脅威と対策

| 脅威 | 確率 | 対策 |
|------|------|------|
| AIがブログトラフィックを奪う | 高 | Guide's Insider Notes + 動画で差別化（実施済み） |
| GetYourGuideが日本市場を支配 | 中 | GYGとの提携（自社ツアーをGYGにも掲載）+ 自社直販の両立 |
| 円安が終わり訪日客減少 | 低 | スペイン語市場の分散投資（実施済み） |
| Manabuの健康問題 | 中 | パッシブ収入（アフィリエイト+デジタル商品）の構築が急務 |
| 競合ガイドがSEOを模倣 | 低 | 先行者利益 + コンテンツ量で障壁構築 |
