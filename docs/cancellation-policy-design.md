# キャンセルポリシーページ設計書

## 1. エグゼクティブサマリー

Tanuki Tabi Travel のキャンセルポリシー専用ページを新設する。
現状は FAQ 内に「悪天候時は全額返金 or リスケ」の一文があるのみで、
予約前の顧客が最も気にする「キャンセル条件」が明示されていない。

本設計書は CEO / CMO / COO / CFO / CSO / トップガイド（Manabu）の
6 視点を統合し、**顧客安心 × 収益保護 × 運用効率 × ブランド強化**
を同時に達成するページを定義する。

---

## 2. 各視点からの要件分析

### 2-1. CEO（経営全体・ブランド信頼）

| 観点 | 要件 |
|------|------|
| ブランド信頼 | 「個人ガイドだから不安」を払拭する明確なポリシー提示 |
| 競合優位 | 大手ツアー会社より柔軟なポリシーを強みに転換 |
| 法的リスク | 特定商取引法・旅行業法に抵触しない表現（※個人ガイドは旅行業法の「旅行業」に該当しない場合が多いが、念のため確認推奨） |
| 顧客LTV | キャンセルではなくリスケを第一選択肢に誘導 → リピート率向上 |

**CEO判断**: ポリシーは「厳格さ」ではなく「透明性と柔軟性」で差別化する。

---

### 2-2. CMO（マーケティング・コンバージョン）

| 観点 | 要件 |
|------|------|
| CVR向上 | 予約フォーム到達前の不安除去。ポリシーページで透明性を示す |
| SEO | `cancellation-policy` ページ自体は検索流入は少ないが、E-E-A-T（信頼性）シグナルとして重要 |
| UXライティング | 法律文書的な堅い文体ではなく、Manabu のフレンドリーなトーンを維持 |
| 導線設計 | FAQ・Contact・Tour Detail の3箇所からリンク。フッターにも常設 |
| 社会的証明 | 「500+ tours completed」「4.86★ rating」をポリシーページにも配置し安心感を補強 |

**CMO判断**: ポリシーページは「安心材料」であり「セールスページの延長」として設計する。

---

### 2-3. COO（オペレーション・運用効率）

| 観点 | 要件 |
|------|------|
| キャンセル受付 | メール (info@tanuki-tabi-travel.com) で受付。フォーム新設は不要（現状の問い合わせ量で十分対応可能） |
| タイムゾーン | 日本時間 (JST/UTC+9) を明記。海外顧客との時差トラブル防止 |
| リスケ運用 | 1回目のリスケは無料。2回目以降は要相談（乱用防止） |
| 悪天候判断 | 気象庁の警報発令を基準に、ガイド側から連絡。判断基準を明文化 |
| エッジケース | 遅刻・ノーショー・途中離脱のルールも明記 |
| 返金手段 | 現状は当日払い（前払いなし）のため、「キャンセル料の請求」は実質発生しない。将来の前払い導入時に備えた設計 |

**COO判断**: 現在の当日払いモデルでは「キャンセル料を請求しない」が最も運用効率が高い。ただし無断キャンセル（ノーショー）への抑止力は必要。

---

### 2-4. CFO（財務・収益保護）

| 観点 | 要件 |
|------|------|
| 機会損失 | 直前キャンセル = その日の売上ゼロ。7日前ルールで代替予約を入れる余地を確保 |
| 前払い未導入 | 現状キャンセル料の回収手段がない → ポリシーは「お願いベース」になる |
| 全額前払い | 競合（Arigato, Tokyo Localized, Magical Trip 等）はほぼ全社が全額前払い。同モデルを採用 |
| ノーショーコスト | ガイドの時間 + 交通費 + 準備時間が無駄になる。ノーショーポリシーで抑止 |

**CFO判断**: 全額前払い + 7日前無料キャンセル + 段階的なキャンセル料で収益を保護。

**収益インパクト試算**:
```
月間ツアー数: 約20件（推定）
平均単価: ¥50,000
全額前払い導入 → ノーショー損失ゼロ化 + 直前キャンセル時の50%回収
年間回収見込み: ¥400,000〜600,000（キャンセル率5%想定）
```

---

### 2-5. CSO（戦略・競合分析）

| 観点 | 要件 |
|------|------|
| 競合比較（プラットフォーム） | Viator/GYG/Airbnb: 24h前（大量・低単価モデル。プラットフォームがリスク吸収） |
| 競合比較（同価格帯） | **Arigato Japan: 7日前** / Sunrise Tours (JTB): 8日前 / Context Travel: 30日前 |
| 競合比較（プレミアムDMC） | Walk Japan: デポジット不可 / Boutique Japan: 最低30%没収 / InsideJapan: 60日+ |
| 差別化ポイント | 「柔軟なリスケ対応」は大手プラットフォームにない強み |
| 市場トレンド | コロナ後、柔軟なキャンセルポリシーが予約決定の重要要因に |
| 前払いの業界標準 | 同価格帯の競合はほぼ全社が「全額前払い」または「デポジット」を採用 |
| 将来拡張 | 複数ガイド体制移行時にもスケールするポリシー構造 |

**CSO判断**: ガイド兼DMC（¥40,000〜80,000/グループ）の適正ラインは「7日前無料キャンセル + 全額前払い」。Arigato Japan・JTB Sunrise Tours と同等で、プレミアムDMCより柔軟な位置づけ。全額前払いは業界標準。

---

### 2-6. トップガイド（Manabu本人）

| 観点 | 要件 |
|------|------|
| 顧客関係 | 旅行中の予定変更はよくあること。柔軟に対応したい |
| 実体験 | 台風・体調不良・フライト遅延での急なキャンセルは実際に発生する |
| 準備コスト | ツアー準備（ルート確認・レストラン予約・資料準備）は前日までに完了。直前キャンセルは準備が無駄になる |
| コミュニケーション | メールでの直接やりとりが最もスムーズ。自動化よりパーソナルな対応を重視 |
| トーン | 法律文書ではなく「友人との約束」のような温かみのある文面にしたい |

**ガイド判断**: お客様の事情を最大限尊重しつつ、お互いの時間を大切にするポリシーにしたい。

## 3. 統合キャンセルポリシー（6視点の合意案）

### 3-0. 競合ベンチマーク（根拠データ）

| 会社 | 無料キャンセル | 中間ティア | 直前 | 前払い |
|------|-------------|-----------|------|--------|
| Arigato Japan | 7日前（-10%手数料） | 4-6日: 50% | 72h以内: 返金なし | 全額前払い |
| Tokyo Localized | 72h前 | 24-72h: 50% | 24h以内: 返金なし | 全額前払い |
| Sunrise Tours (JTB) | 8日前 | 7-2日: 30%, 前日: 40%, 当日: 50% | ノーショー: 100% | 10%デポジット |
| Context Travel | 30日前 | 29日-48h: 50% | 48h以内: 返金なし | 全額前払い |
| ToursByLocals | 15日前 | — | 15日以内: 返金なし | 全額前払い |

**Tanuki Tabi Travel のポジショニング**: Arigato Japan・JTB Sunrise Tours と同等の「7日前」を基準に、3段階のティアを設定。

### 3-1. キャンセルタイムライン

| タイミング | 対応 | 根拠 |
|-----------|------|------|
| **7日以上前** | 全額返金（無料キャンセル） | CSO: Arigato Japan(7日)・JTB(8日)と同等。CMO: 「7-day free cancellation」は十分な訴求力 |
| **3〜6日前** | 50%返金 | CFO: 準備コストの一部回収。CSO: Arigato(4-6日50%)と整合 |
| **72時間以内** | 返金なし（リスケを強く推奨） | COO: 準備完了後の損失防止。ガイド: やむを得ない事情は個別対応 |
| **ノーショー（無断不参加）** | 返金なし | CFO: 全額損失の抑止。COO: 公平性 |

### 3-1b. 支払い方式: 全額前払い

| 項目 | 内容 |
|------|------|
| 支払い額 | ツアー料金の全額 |
| 支払いタイミング | 予約確定時 |
| 返金方法 | 元の支払い方法に返金（7〜14営業日） |
| 返金額 | キャンセルタイムライン（3-1）に準拠 |

> 競合参考: Arigato Japan・Tokyo Localized・Magical Trip はいずれも全額前払い。業界標準に準拠。

### 3-2. ガイド側からのキャンセル

ガイド都合でのキャンセルは、いかなる場合も顧客に不利益を生じさせない。

| 状況 | 判断基準 | 対応 |
|------|---------|------|
| **悪天候** | 気象庁の警報（暴風・大雨・洪水等）発令時 | 全額返金 or リスケ（顧客選択） |
| **ガイドの体調不良・怪我** | ツアー遂行が困難とガイドが判断した場合 | 全額返金 or リスケ（顧客選択） |
| **自然災害** | 地震・台風・噴火等で安全が確保できない場合 | 全額返金 or リスケ（顧客選択） |
| **公共交通の大規模運休** | ツアールートの主要交通機関が運休した場合 | 全額返金 or リスケ（顧客選択）。代替ルート提案も可 |
| **その他ガイドの個人的事情** | 家族の緊急事態等 | 全額返金 or リスケ（顧客選択） |

- ガイド側キャンセルの連絡は **判明次第、速やかに** メールで行う
- リスケの場合、顧客の滞在中に代替日程を最優先で調整する
- いずれの場合も **全額返金を保証**。顧客に追加コストは発生しない

### 3-2b. やむを得ない事情による顧客キャンセル（Exceptional Circumstances）

通常のキャンセルタイムライン（3-1）にかかわらず、以下の事情が確認できる場合は柔軟に対応する。

| 事情 | 確認方法 | 対応 |
|------|---------|------|
| **本人または同行者の急病・怪我** | 医師の診断書、または宿泊先からの連絡 | 全額返金 or リスケ |
| **フライトの大幅遅延・欠航** | 航空会社の通知メール、搭乗券の提示 | 全額返金 or リスケ |
| **入国トラブル** | パスポート・ビザ関連で入国できない場合 | 全額返金 |
| **近親者の死去・重篤な病気** | 状況に応じて柔軟に判断 | 全額返金 or リスケ |
| **渡航先の安全上の問題** | 外務省・大使館の渡航警告レベル引き上げ | 全額返金 or リスケ |

- やむを得ない事情に該当するかは **ガイド（Manabu）が個別に判断**
- 証明書類の提出は必須ではないが、状況確認のために依頼する場合がある
- 判断に迷う場合は **顧客有利の原則** で対応する

### 3-3. リスケジュールポリシー

- 1回目: 無料。滞在中の別日程に変更可能
- 2回目以降: 可能な限り対応するが、確約はしない
- リスケ依頼もメールで受付

### 3-4. 遅刻・途中離脱

| 状況 | 対応 |
|------|------|
| 顧客の遅刻（15分以内） | ツアー時間を調整して対応 |
| 顧客の遅刻（15分超・連絡なし） | ノーショー扱い |
| 顧客の途中離脱 | 残り時間分の返金なし（ツアーはその時点で終了） |
