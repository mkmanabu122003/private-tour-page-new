# Pexels画像 推奨リスト

## 必要な画像とPexels検索キーワード

### 1. 浅草（Asakusa）

#### ヒーロー画像用（横長）
**検索URL:** https://www.pexels.com/search/sensoji%20temple/
**推奨キーワード:** "sensoji temple lantern", "kaminarimon gate", "asakusa night"
**探すポイント:**
- 雷門（大提灯）が中心の横長写真
- 夕方や夜のライトアップ写真が印象的
- 人が少なめの写真が使いやすい

#### ツアーカード用（正方形/縦長）
**検索URL:** https://www.pexels.com/search/asakusa/
**推奨キーワード:** "sensoji temple", "nakamise street"
**探すポイント:**
- 浅草寺本堂や五重塔
- 仲見世通りの賑やかな様子
- スカイツリーと浅草の組み合わせ

## ダウンロード手順

### スマホから：

1. **上記のURLをタップ**
2. **気に入った写真をタップ**
3. **「Free Download」ボタンを長押し**
4. **「リンクをコピー」を選択**
5. **コピーしたURLをClaude（私）に送信**

### PCから：

1. **上記のURLをクリック**
2. **気に入った写真をクリック**
3. **「Free Download」ボタンを右クリック**
4. **「リンクのアドレスをコピー」を選択**
5. **コピーしたURLをClaude（私）に送信**

## URL形式の例

正しいURL形式：
```
https://images.pexels.com/photos/123456/pexels-photo-123456.jpeg?auto=compress&cs=tinysrgb&w=1920
```

または写真ページのURL：
```
https://www.pexels.com/photo/sensoji-temple-tokyo-123456/
```

どちらでも大丈夫です！

## 推奨画像サイズ

- **ヒーロー画像**: Large (1920px) または Original
- **ツアーカード**: Medium (800px) または Large (1920px)

サイズが大きすぎても、ビルド時に自動的に最適化されます。

---

## 画像選びのコツ

1. **明るくクリアな写真**を選ぶ
2. **人が写りすぎていない**写真が使いやすい
3. **天気が良い日**の写真が印象的
4. **構図がシンプル**な写真が読みやすい
5. **有名なランドマーク**が分かりやすく写っている
